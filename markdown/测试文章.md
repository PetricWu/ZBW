# 测试文章标题

这是一篇测试文章，用于验证文章列表更新功能。

## 功能说明

当用户上传新的 Markdown 文件后，需要重新运行 `generate_index.py` 脚本来更新 `index.json` 文件。

文章系统的工作流程：
1. 上传 Markdown 文件到 `markdown/` 目录
2. 运行 `python generate_index.py` 生成/更新 `index.json`
3. 前端页面通过 `fetch('/markdown/index.json')` 加载文章列表
4. 用户可以点击文章标题跳转到文章详情页

## 测试步骤

1. 创建新的 Markdown 文件
2. 运行 `python generate_index.py`
3. 刷新首页，查看文章列表是否更新
4. 点击文章标题，验证文章详情页是否正常显示

## 预期结果

- 首页文章列表应该显示新上传的文章
- 点击文章标题应该能正确跳转到文章详情页
- 文章详情页应该能正确渲染 Markdown 内容
