# 招标代理服务流程详解

招标代理服务是保障招标活动规范、高效进行的重要环节。本文将详细介绍招标代理服务的完整流程。

## 一、项目准备阶段

### 1.1 接受委托

- 与采购人签订委托代理合同
- 明确双方权利义务
- 确定代理服务范围

### 1.2 资料收集

- 收集项目基本情况
- 了解采购需求和技术要求
- 获取相关审批文件

## 二、招标文件编制

### 2.1 编制招标文件

招标文件应当包括以下内容：

1. **招标公告**（或投标邀请书）
2. **投标人须知**：说明投标文件的编制要求和提交程序
3. **评标办法**：明确评标标准和方法
4. **合同主要条款**：包括合同通用条款和专用条款
5. **技术规格和要求**：详细说明采购货物或服务的规格
6. **投标文件格式**：统一投标文件的格式要求

### 2.2 审核确认

- 提交采购人审核
- 根据意见进行修改
- 最终确认定稿

## 三、招标公告发布

### 3.1 发布渠道

- 政府采购网
- 公共资源交易中心网站
- 行业媒体
- 其他指定媒介

### 3.2 公告内容

- 采购人及采购代理机构信息
- 项目名称及编号
- 预算金额
- 获取招标文件的方式和时间
- 投标截止时间
- 开标时间和地点

## 四、投标人报名及答疑

### 4.1 投标人报名

- 接受投标人报名
- 发放招标文件
- 记录投标人信息

### 4.2 答疑澄清

- 收集投标人疑问
- 组织现场踏勘（如有需要）
- 发布澄清或修改公告

## 五、开标评标

### 5.1 开标程序

1. 检查投标文件密封情况
2. 宣布开标纪律
3. 当众拆封投标文件
4. 宣读投标报价等内容
5. 记录开标情况

### 5.2 评标程序

1. **组建评标委员会**：依法组建评标委员会
2. **初步评审**：审查投标文件是否实质性响应
3. **详细评审**：按照评标办法进行打分
4. **编写评标报告**：提出推荐意见

## 六、定标及公示

### 6.1 定标

- 采购人根据评标报告确定中标人
- 发布中标公告
- 发出中标通知书

### 6.2 公示期限

- 中标公告公示期一般为3个工作日
- 接受社会监督

## 七、签订合同

### 7.1 合同签订

- 中标通知书发出后30日内签订书面合同
- 合同内容与招标文件和中标文件一致
- 办理合同备案

### 7.2 履约监督

- 监督合同履行情况
- 处理合同履行中的问题
- 组织项目验收

## 八、资料归档

### 8.1 资料整理

- 整理招标过程所有文件
- 建立完整的项目档案
- 保存期限不少于15年

### 8.2 归档内容

- 委托代理合同
- 招标文件及澄清文件
- 投标文件
- 开标评标记录
- 评标报告
- 中标通知书
- 采购合同

## 九、服务流程图

```text
项目准备 → 文件编制 → 公告发布 → 报名答疑 → 开标评标 → 定标公示 → 签订合同 → 资料归档
```

## 十、注意事项

1. **严格遵守时间规定**：各个环节必须按时完成
2. **保证信息透明**：及时发布相关信息，接受社会监督
3. **维护公平竞争**：平等对待所有投标人，不得歧视
4. **做好保密工作**：妥善保管评标过程资料，不得泄露

## 十一、常见问题解答

### Q1: 招标文件可以修改吗？

A: 可以修改，但必须在投标截止时间15日前发布澄清或修改公告，否则应当顺延投标截止时间。

### Q2: 投标人少于三家怎么办？

A: 根据规定，投标人不足三家的，应当废标，重新组织招标。

### Q3: 评标委员会如何组成？

A: 评标委员会由采购人代表和有关技术、经济等方面的专家组成，成员人数为5人以上单数，其中专家人数不得少于成员总数的三分之二。

---

通过规范的招标代理服务流程，可以有效保证招标活动的公平、公正、公开，提高采购效率，节约采购成本。
