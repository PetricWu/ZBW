# 关于2024年度政府采购项目的招标公告

根据《中华人民共和国政府采购法》等相关法律法规，现就2024年度政府采购项目进行公开招标，欢迎符合条件的供应商参与投标。

## 一、项目基本情况

1. **项目名称**：2024年度政府采购项目
2. **采购方式**：公开招标
3. **预算金额**：500万元
4. **采购内容**：办公设备采购及相关服务

## 二、投标人资格要求

1. 具有独立承担民事责任的能力；
2. 具有良好的商业信誉和健全的财务会计制度；
3. 具有履行合同所必需的设备和专业技术能力；
4. 有依法缴纳税收和社会保障资金的良好记录；
5. 参加政府采购活动前三年内，在经营活动中没有重大违法记录。

## 三、获取招标文件

1. **时间**：2024年1月15日至2024年1月25日（每日9:00-17:00）
2. **地点**：XX市公共资源交易中心
3. **方式**：现场领取或网上下载
4. **售价**：免费

## 四、投标截止时间

2024年2月5日9时30分

## 五、联系方式

联系人：张先生
联系电话：400-888-8888
联系地址：北京市朝阳区

## 六、其他事项

1. 本项目不接受联合体投标；
2. 逾期送达的或者未送达指定地点的投标文件，采购人不予受理；
3. 本次招标公告在政府采购网、公共资源交易中心网站等媒介发布。
