## 🐛 常见问题排查

### 页面显示 404

**可能原因**：
- 文件路径错误
- 大小写不匹配（Linux 区分大小写）
- 文件未推送到 GitHub

**解决方法**：
1. 检查 GitHub 仓库是否有该文件
2. 检查链接路径是否正确
3. 检查文件名大小写是否一致

### 图片无法显示

**可能原因**：
- 图片文件未推送到 GitHub
- 图片路径错误
- 图片格式不支持

**解决方法**：
1. 检查 `images/` 文件夹是否已推送
2. 检查图片路径是否正确
3. 尝试使用常见格式（JPG、PNG、WebP）

### 样式未加载

**可能原因**：
- CSS 文件路径错误
- CSS 文件未推送到 GitHub
- 浏览器缓存

**解决方法**：
1. 检查 `<link>` 标签路径是否正确
2. 硬刷新浏览器（Ctrl+F5 或 Cmd+Shift+R）
3. 清除浏览器缓存

### 自动部署不触发

**可能原因**：
- Netlify/Vercel 未正确连接 GitHub
- 推送的分支不匹配
- 仓库设置问题

**解决方法**：
1. 检查 Netlify/Vercel 的 GitHub 集成状态
2. 确认推送到正确的分支（main/master）
3. 检查仓库设置
